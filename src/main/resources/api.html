PurrBotAPI - An image-api for everyone!

This API was made to provide dynamic images for *Purr*
It acts as a HTTPS/REST API.
All generated images are NOT stored. They are generated "on the fly" through link parameters.
See the sections below for available endpoints and parameters.

This API is version-less, which means, that it doesn't depend on you using something like v2 in the link.
Below is the base-url, that you can use for connecting to the endpoints.

Base-URL:

        https://purrbot.site/api

Endpoints as JSON

        https://purrbot.site/api.json

GET /quote

Returns a image that looks like a message from a discord-user.

Parameters

Field Type Description
avatar URL avatar-url
color String/Integer Name color. Either hex:rrggbb, rgb:r,g,b or a raw color as integer value
format String time format
name String Username
text String Message
time Integer Time in epoch-millis

GET /status

Returns a user-avatar with a status-icon.

Parameters

Field Type Description
avatar URL avatar-url
status String Status for the icon (online, idle, do_not_disturb, dnd or offline)

GET /welcome

Returns awelcome image with the users avatar, name and guild-count.

Parameters

Field Type Description
avatar URL avatar-url
color String/Integer Text color. Either hex:rrggbb, rgb:r,g,b or a raw color as integer value
image String The welcome-image
name String Username
size Integer Member count of guild
